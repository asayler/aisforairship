A is for Airship
================

An Engineer's Blog

By [Andy Sayler](https://www.andysayler.com)

# Prerequisites
* [hugo](https://gohugo.io/) (extended edition)
* [golang](https://go.dev/)
* git

# Development
1. `$ hugo server -D`

# Build
1. `$ rm -rf ./public`
2. `$ HUGO_ENV=production hugo`
